# Overview
- This project is a real-time system that tracks and reads vehicle license plates. It uses two main tools to work fast and accurately:
  + ByteTrack (Tracking): This tool tracks the license plates smoothly as they move in the video. It is very fast and keeps the system running in real-time.
  + PaddleOCR (Reading Text): This tool reads the letters and numbers on the plates. We used transfer learning to train this model, making it much better and more accurate at reading.
